# 查看操作日志

SAE记录了所有应用及资源相关的操作详情，包括操作时间、操作内容、操作人ID等信息。您可以通过这些信息追踪应用和资源的变更。本文介绍如何查看操作详情。

## 操作步骤

1.  登录[SAE控制台](https://sae.console.aliyun.com)。

2.  在左侧导航栏单击**应用列表**，在**应用列表**页面上方选择**地域**，单击具体应用名称。

3.  在左侧导航栏中选择**限流降级（仅支持Java）** \> **操作日志**。

4.  在**操作日志**页面，查看操作详情。
